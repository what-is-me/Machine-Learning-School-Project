- LLE
- PCA

- K-means++

- [matplotlib](https://github.com/matplotlib/matplotlib)
- [tqdm](https://github.com/tqdm/tqdm)
- [scikit-learn](https://github.com/scikit-learn/scikit-learn)仅使用NearestNeighbors部分
